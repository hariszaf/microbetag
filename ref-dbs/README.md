# Data and databases used in `microbetag` 




Currently, `microbetag` exploits genomes from the following databases: 
* KEGG
* GTDB
* MGnify catalogues

| Resource | unique ncbi ids added |
|:--------:|:----------------------|
| GTDB     | 14,818                |
| KEGG     | 3,984                 |
| MGnify   | 1,044
